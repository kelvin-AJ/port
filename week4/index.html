  • Forms
    1. The span attribute is the URL where the form data would be submitted
    2. We can access form element by using document.forms which returns an HTML collection of all forms in the document. we can also use document.getElementsByTagname("form")[0]
    3. we can use the name attribute to identify a form. We can do this with both dot "." and bracket "[]" notation. Using the bracket notation is more advised to avoid selection form properties.
    4. the elements method returns an HTML collection of all elements contained in the form.
    5. We can also access the form controls using their name attributes as if it was a property of the form object. so if set the name attribute of a first Name input to firstName, we can access it like form.firstName
    6. form.submit() would submit the form automatically. A form can be submitted manually by using an actual button or an input type of submit.
    7. form.reset() resets all form controls back to their initial values. a button type with the attribute of reset does the same thing. Reset buttons are generally NOT adviced. You don't want the users to wipe out all the data they entered when it's not yet saved.
    8. form.action sets the action attribute of a form. (form events in console)

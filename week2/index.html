  • Programming Basics:
    1. Primitive Data Types include: String, Symbol(ES6), Number, Boolean, Undefined, Null. arrays, functions, abject literals are objects
    2. An operator applies operation to a value. A binary operator requires two operands
    3. When defining variables, 'const' for values that wouldn't be changed. 'let' for values that can be reassigned. const and let are block scoped.
    4. A scope basically everywhere a variable/constant can be accessed.
      • Global Scope: accessible everywhere
      • Local Scope: accessible only locally(inside the block)
    5. camelCase is probably the best variable naming convention to use in JS
    6. Square brackets are an alternative to accessing primitive data types: name['length']
    7. Some string formatting methods:
      name.toUpperCase(); , name.toLowerCase(); , name.charAt(1);, name.indexOf('x'); and so on. We also surround strings with backticks like this `This "string" is surrounded in backticks so i can use normal quotations(") inside and i can insert variable and other values using this ${myVariable}`
    8. Then we also have ES6 methods for Numbers like (), Number.isFinite(). Then methods that come with numbers like toExponential(), toFixed(), toPrecision()
    9. Type coercion is a smart feature of JS but can make it difficult to spot errors. Avoid it
    10. We can convert Strings to Numbers and vice versa: Number('3') and String(6). This is especially useful when working with user inputs that are strings. To 'Parse' to integers or Numbers we use parseInt('string', number base to be parsed to e.g 10 is to base 10) or parseFloat('string', number base to be parsed to e.g 10 is to base 10)
  • Arrays, Logic and Loops:
    1. To create arrays we can use square brackets of the array constructor function.
    2. Destructuring arrays invloves taking out values from an array and using them as individual values
    3. Array methods: (example of use in console)
      • The Array.prototype.slice() method returns an array "chopped" off from another.
      • The Array.prototype.splice() method deletes elements from and array. It can also be used to replace elements in the array.
      • The Array.prototype.sort() method sorts strings and numbers alphabetically.
      • The Array.prototype.indexOf() method gets the index of an element
      • The Array.prototype.includes() method returns a boolean (true/false) for if the element is included in the array.
    4. We can have mulitdemntional array too [[1,4],[5,6]]
    5. Sets are data structures used to store unique values.We create a new Set by using the constuctor new Set()
    6. We can convert array to sets and by passing in the array as a parameneter to the constructor. Sets also have methods for adding, deleting and other modifictaion methods.
  • Functions:
    • We can declare functions by using function declarations:
      function myFunction() {
      --what the function does goes here--
      }
      or function expressions:
      const myFunction = function() {
      --what the function does goes here--
      }
      we also have arrow functions.
    • Whatever we return from a function using the return keyword, becomes the value we get when the function is called.
    • Arguments we pass into functions are accesible by the function and the function uses the arguements to return desired values.
    • We can assign deafault values to srguements by using the assignment operator with the arguments.
    • Hoisting invloves invoking(calling) functiosn before they are defined.
    • A function ecomes a callback funcion when it is passed as a parameter of another function. Array iterators like, forEach, map, filter, reduce and so on, take in callback functions. In most situations, arrow functions are used.
    • We can chain array iterators together because they all return arrays.
